Podemos observar que tenemos 5 campos con datos faltantes en el dataset. El dataset cuenta con un total de 1048575 de registros. Como vemos en la tabla, los campos con datos faltantes son: Intubados, neumonia, edad, embarazo y terapita_intensiva. De estos campos, Intubados y terapia_intensiva tienen un 81% de datos faltantes y embarazos 49%. Por lo tanto, se eliminaran del dataset por que tienen + de 30% de datos nulos y por lo tanto evitar afectar la informacion. Para el resto de los campos con nulos se procedera a eliminar la fila que contenga el dato faltante.

Creamos el campo fallecido¶

El campo Date died indica mediante la fecha si el paciente murio y si no murio pone 9999-99-99. Creamos un campo de fallecido que sera 1 si murio y 2 si no murio. Esto permitira una mejor lectura y un mejor uso de los datos en el momento de realizar el modelo de prediccion.

In [ ]:
# creamos el campo fallecidos a partir del campo date died. Si hay fecha el paciente fallecio, si la fecha es 9999-99-99 el paciente no fallecio.

df_covid['fallecidos'] = df_covid['DATE_DIED'].apply(lambda x: 0 if x == '9999-99-99' else 1)
